## 自定义包的注意点
1. 包的名字不要随意. 不要写 abc 或者 test. albumer-rand-function
2. 包的名字不能重复. (paramserver-random-function) 包在仓库中的名字是唯一的.

## NPM 命令行只需要登录一次

## 包的更新
一定要注意修改版本号.

## 包的删除
```
npm unpublish 包名 --force
```

## CNPM 的注意
安装包的时候, 一定要加 --save(-S) 或者 --save-dev(-D)
